# 🎬 Saints Films – Web Application

## Professional Audiovisual Company Website

Saints Films is a modern web application developed to represent a Brazilian audiovisual production company, focused on delivering a strong digital presence with smooth animations, structured navigation, and scalable frontend architecture.

The project aims to elevate the company’s online identity, showcasing its services, brand, and professionalism through a clean and engaging user experience.

---

## Objective

Provide a high-quality institutional website to:

- Strengthen the company’s digital presence
- Showcase audiovisual services and brand identity
- Deliver a modern and engaging user experience
- Serve as a scalable base for future features (e.g. live chat, backend integration)

---

## Core Features

- Single Page Application (SPA) architecture
- Smooth animations with Framer Motion
- Page navigation using React Router
- Clean and modern UI design
- Responsive layout for multiple devices
- Component-based structure for scalability
